Guia rapida

Visor fotografico ByAlaitz

Una pantalla limpia para enseñar las fotos del evento en bucle, con carga desde Google Drive y controles sencillos desde el panel de administracion.

Enlaces

Donde entrar

  • Visor publico: https://anderibanez123.github.io/ByAlaitz_VisorFotografico/
  • Panel admin: /admin/ al final de la URL.
  • Clave del panel: stand2026

Tambien puedes entrar al admin tocando el logo pequeno de la esquina inferior derecha del visor.

Uso diario

Subir fotos

  1. 1Abre la carpeta de Google Drive preparada para el visor.
  2. 2Sube las fotos desde movil, tablet u ordenador.
  3. 3Espera unos segundos. La web revisa la carpeta automaticamente.
  4. 4Las imagenes nuevas aparecen en la presentacion sin tocar codigo.
Presentacion

Como dejarlo funcionando en el stand

  1. 1Abre el visor publico en Chrome, Edge o Safari.
  2. 2Haz doble click o doble toque sobre la foto para entrar en pantalla completa. Tambien puedes usar el boton FS.
  3. 3Para salir de pantalla completa, pulsa Esc.
  4. 4Si necesitas ajustar algo, toca el logo de abajo a la derecha y entra al admin.
Panel admin

Opciones principales

  • Tiempo entre fotos.
  • Velocidad de transicion.
  • Efectos: fundido, deslizamiento, zoom, barrido, desenfoque, giro, volteo 3D o corte directo.
  • Orden secuencial o aleatorio.
  • Fondo decorativo activado o desactivado.
  • Frecuencia de actualizacion de Google Drive.
Barra superior

Redes y contacto

  • Puede estar fija o deslizante.
  • Se puede cambiar la velocidad del movimiento.
  • Puedes activar, quitar o anadir Instagram, TikTok, correo, telefono, web u otra red.
  • En pantalla se muestra como icono + dato, discreto y limpio.
Guardado

Que se guarda

Los cambios del admin se guardan automaticamente en el navegador. Si se quiere dejar una configuracion fija para todo el mundo, se puede usar Exportar config.json y subir ese archivo al repositorio.

Si algo falla

Comprobaciones rapidas

  • Si no salen fotos, revisa que la carpeta de Drive tenga imagenes directamente dentro.
  • Si Drive no carga, revisa que la API key siga activa y permitida para la web.
  • Si un cambio no aparece, refresca con Ctrl + F5.
  • Si el QR no deja subir, abre la carpeta con una cuenta que tenga permiso de edicion.
